I am a very simple card. I am good at containing small bits of information. I am convenient because I require little markup to use effectively.

  • 1: Required data format
  • 2: The upload process

The file format needs to be some form of text-based file (like a CSV file) with each entry on a line and seperated by one of the supported delimiters


If you have an Excel-based formatted file then either use Excel or an alternative to export the data to a CSV or use csvkit to convert it from command line


Before starting you should ensure that the data matches the tempate at the link below


Data template CSVKIT

A. File:
Click choose file and navigate to the file you want to upload.
Note: there may be issues when trying to upload files larger than 20MB


B. Header:
If your file has a line at the top with titles for the columns then this is referred to as the header and this line will need to be ignored when importing the data.
If this line is present in your file then ensure that the checkbox is ticked off (it is by default)


C. Delimiter:
The delimiter is the character that goes between fields in each entry in the file. Normal CSV files use either a comma or semicolon.
Select the appropriate delimiter for your file from the list
If your delimiter is not in the list you can either contact one of the team to have it added or you can use a text editor to 'find and replace' to replace yor delimiter with a supported one


D. Upload:
The final step for you is to click on the Upload button which will transfer the file and begin processing through it.
Please be very patient as it can take a few minutes to process and clean up large portions of data so if it seems like it is loading for a long time it is probably still busy


E. Confirmation:
The last thing that will happen is a message will be displayed at the top of the screen detailing the number of entries that were successfully added and the number that failed to be added
